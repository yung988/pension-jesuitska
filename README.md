# Pension Jesuitska

Web pro pension Jesuitska vytvořený pomocí Next.js, React 19, TypeScript a TailwindCSS.

## Funkce

- Responzivní design
- Moderní UI s využitím TailwindCSS
- Optimalizovaná výkonnost
- SEO přívětivý

## Technologie

- Next.js 15
- React 19
- TypeScript
- TailwindCSS
- Shadcn/ui komponenty
- Embla Carousel

## Vývoj

```bash
# Instalace závislostí
pnpm install

# Spuštění vývojového serveru
pnpm dev
```

## Build

```bash
# Vytvoření produkčního buildu
pnpm build

# Spuštění produkční verze
pnpm start
```
